Selamat Datang di DataApp

Aplikasi sederhana untuk menyimpan dan mengelola data penting. Data disimpan di cloud sehingga dapat diakses oleh siapa saja yang menggunakan aplikasi ini.

Fitur Utama

Penyimpanan Data di Cloud

Data disimpan di Firebase dengan keamanan tinggi dan dapat diakses dari mana saja.

Akses Universal

Data dapat diakses oleh semua pengguna yang terdaftar dalam aplikasi.

Pencarian Cepat

Temukan data dengan cepat menggunakan fitur pencarian yang mudah digunakan.

Tombol Salin

Salin data dengan tepat menggunakan tombol salin yang telah disediakan.
